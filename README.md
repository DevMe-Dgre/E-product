# E-product

This is a little proejct for the E-commerce with the products listed on the site having the prices mentioned with the reviews along with the old priced badging crossed out making them listed as the sales price. 

Here is the live-demo link for this product: e-product-beta.vercel.app

Tech stack used : HTML/CSS and javascript

Built the cards for each of the products, by using the HTML. Styled them using CSS in the grid stucture. And used JAVASCRIPT for to add the product to the cart.
